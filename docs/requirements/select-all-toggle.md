# 전체 선택 토글 요건 (alt+u)

## 개요

파일 목록에서 `alt+u` 단축키로 항목을 일괄 선택/해제한다.
**3단계 순환 토글**로 동작하며, ext 유형이 `[드라이브]`인 항목은 모든 단계에서 제외한다.

---

## 3단계 토글 동작

| 단계 | 동작 | 포함 대상 |
|------|------|-----------|
| 1단계 | 노멀 파일 선택 | 파일 (숨김 파일 표시 중이면 숨김 파일도 포함), 폴더 제외 |
| 2단계 | 파일 + 폴더 선택 | 1단계 대상 전체 + 폴더 |
| 3단계 | 전체 선택 해제 | — |

다음 `alt+u`는 다시 1단계로 돌아간다.

---

## 공통 제외 조건

- ext 유형이 `[드라이브]`인 항목은 **모든 단계에서 선택 대상에서 제외**한다.

---

## 숨김 파일 포함 기준

- 숨김 파일 표시 여부는 현재 뷰 설정을 따른다.
- 숨김 파일이 **보이는 상태**라면 1단계부터 선택에 포함한다.
- 숨김 파일이 **숨겨진 상태**라면 선택 대상에서 제외한다.

---

## 단축키

| 키 | 동작 |
|----|------|
| `alt+u` | 3단계 순환 토글 |
